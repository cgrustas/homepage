# Homepage

## About This Project

A responsive portfolio homepage, built as part of [The Odin Project](https://www.theodinproject.com/). The webpage matches design templates for for three different viewport sizes: mobile, tablet, and desktop.

## Skills Learned

### Responsive Design Fundamentals

- Implemented mobile-first development approach to prioritize content flow
- Created media queries to adapt layouts between common breakpoint widths

### CSS Techniques

- Applied float for the 'about' section layout in the tablet view
- Used translate to reposition elements while maintaining their original place in the document flow

### Image Handling & Optimization

- Configured images to scale properly within containers using CSS
- Implemented `<picture>` element with multiple source sets for different screen sizes
- Learned aspect ratio preservation techniques for responsive images
